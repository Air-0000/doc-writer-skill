# 格式规范库

> 本目录收录各类型文档的格式规范，Writer/Optimizer/Reviewer 在写作和审查时参考。

## 目录

| 文件 | 适用场景 |
|------|---------|
| [academic_paper.md](academic_paper.md) | 中文学术论文（期刊/学位论文/会议论文）|
| [project_proposal.md](project_proposal.md) | 项目申报书/可行性报告 |
| [novelty_report.md](novelty_report.md) | 科技查新报告 |
| [business_plan.md](business_plan.md) | 商业计划书 / BP |
| [technical_doc.md](technical_doc.md) | 技术方案文档 / 系统设计文档 |

## 更新规则

- 每季度更新一次格式规范
- 用户反馈格式错误后，24小时内更新对应文件
- 新增文档类型时，在本 index.md 中追加对应条目

## 使用方式

1. Writer 开始写作前，读取对应文档类型的 format_specs/*.md
2. Reviewer 审查时，对照 format_specs 检查格式合规性
3. 用户指定文档类型时，Planner 自动读取对应的格式规范文件
