# 学术论文格式规范

## 中文学术论文标准结构

1. **题目**（Title）：简洁、准确，≤25字，含关键词
2. **摘要**（Abstract）：300-500字，含目的/方法/结果/结论
3. **关键词**（Keywords）：3-8个，优选《中文图书分类法》词
4. **引言**（Introduction）：研究背景→国内外现状→研究目的→创新点
5. **相关工作/文献综述**（Related Work）
6. **方法/原理**（Method）
7. **实验/结果**（Experiment & Results）
8. **讨论**（Discussion）
9. **结论**（Conclusion）
10. **参考文献**（References）

## 格式要求

| 项目 | 要求 |
|------|------|
| 正文字体 | 宋体/Times New Roman，小四/12pt |
| 行距 | 1.5倍行距 |
| 段落首行 | 首行缩进2字符 |
| 标题层级 | 一级黑体三号，二级黑体四号，三级宋体小四 |
| 图表标题 | 图在下方，表在上方，编号连续 |
| 参考文献 | GB/T 7714-2015 格式 |

## 常见错误

- [错误] 摘要写成引言 → 正确做法：摘要应包含目的、方法、结果、结论四要素
- [错误] 图表标题过于简单 → 正确做法：图表标题应说明"是什么"而非"如图所示"
- [错误] 参考文献格式不统一 → 正确做法：严格按 GB/T 7714 格式，逐条核对
- [错误] 创新点表述模糊 → 正确做法：明确说明与已有方法的差异和创新
